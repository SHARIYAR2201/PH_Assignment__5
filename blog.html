What are the different ways to select an element in the DOM?

Elements can be selected by ID, class name, tag name, attribute, or query selectors.

What is the difference between innerHTML, innerText, and textContent?

innerHTML: Retrieves or sets HTML content.
innerText: Retrieves or sets only visible text.
textContent: Retrieves or sets all text, including hidden elements.

What is event delegation in the DOM?

Event delegation is a technique where an event listener is attached to a parent element instead of multiple child elements.

What is event bubbling in the DOM?

Event bubbling allows an event to start from the target element and propagate upwards through its ancestors in the DOM tree.

How do you create, add, and remove elements using JavaScript?

Create: document.createElement("tag").
Add: parent.appendChild(element).
Remove: parent.removeChild(element).
